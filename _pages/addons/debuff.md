**Author:** atom0s<br/>

Allows players to cancel buffs/debuffs via a slash command.

## Features

  * Cancels buffs/debuffs via a command.

## Commands

### debuff, cancel
Syntax:
```
/debuff <id>
/cancel <id>
```
  * Cancels the given status id.
