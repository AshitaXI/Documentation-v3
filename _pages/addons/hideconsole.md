**Author:** atom0s<br/>

Hides the boot loader window used for private servers.

## Features

  * Hides the private server boot loader window.

## Commands

### hideconsole
Syntax:
```
/hideconsole show
/hideconsole hide
```
  * Shows the boot loader window.
  * Hides the boot loader window.
